## 🎯 نظرة عامة | Competition Overview

### المفهوم | Concept
مسابقة تكنولوجية شاملة تربط بين التراث المصري القديم والتكنولوجيا الحديثة، حيث يستخدم الطلاب مهاراتهم التقنية لإعادة تخيل وتقديم الحضارة المصرية بطرق مبتكرة.

A comprehensive technology competition linking Ancient Egyptian heritage with modern technology, where students use their technical skills to reimagine and present Egyptian civilization in innovative ways.

### الفئات المستهدفة | Target Categories

**المستوى الأول | Level 1:** الصف الأول والثاني (مهارات تقنية أساسية)
- **Grades 1-2:** Basic technology skills

**المستوى الثاني | Level 2:** الصف الثالث والرابع والخامس (برمجة Scratch)
- **Grades 3-5:** Scratch programming

**المستوى الثالث | Level 3:** المرحلة الإعدادية (برمجة Python)
- **Preparatory Stage:** Python programming

## 📋 المهام التفصيلية | Detailed Tasks

### المستوى الأول: الصفوف 1-2 | Level 1: Grades 1-2

#### المهمة الرئيسية | Main Task
**"رحلة الفرعون | The Pharaoh's Journey"**

إنشاء مشروع إبداعي يوضح رحلة فرعون في مصر القديمة باستخدام:
- الرسم والتلوين
- الكولاج الرقمي (استخدام تطبيقات بسيطة)
- اللعب بالمنطق (ألغاز بسيطة)
- الأعمال اليدوية التقنية

Create a creative project showing a Pharaoh's journey in Ancient Egypt using:
- Drawing and coloring
- Digital collage (using simple apps)
- Logic play (simple puzzles)
- Tech-based crafts

#### المتطلبات | Requirements
- **الفرق**: 2-3 طلاب | **Teams**: 2-3 students
- **الوقت**: أسبوعان للتحضير | **Time**: 2 weeks preparation
- **الأدوات المسموحة**: أقلام، ألوان، تطبيقات الرسم البسيطة (Paint, Tux Paint, etc.)
- **Allowed Tools**: Pens, colors, simple drawing apps

#### أمثلة للمخرجات | Example Outputs
1. لوحة رسم كبيرة تصور رحلة عبر النيل | Large drawing depicting Nile journey
2. كولاج رقمي يجمع صور الأهرامات والمعابد | Digital collage combining pyramids and temples
3. لغز بسيط (puzzle) يمكن للآخرين حله | Simple puzzle others can solve
4. نموذج ثلاثي الأبعاد من الورق المقوى | 3D paper model

### المستوى الثاني: الصفوف 3-5 | Level 2: Grades 3-5

#### المهمة الرئيسية | Main Task
**"أسرار الأهرامات التفاعلية | Interactive Pyramid Secrets"**

إنشاء لعبة أو قصة تفاعلية أو رسوم متحركة في Scratch حول مغامرة في مصر القديمة.

Create a game, interactive story, or animation in Scratch about an adventure in Ancient Egypt.

#### المتطلبات الإلزامية | Mandatory Requirements

**العناصر البرمجية المطلوبة | Required Programming Elements:**
1. ✅ 3 شخصيات على الأقل (Sprites) | At least 3 sprites
2. ✅ خلفيتان مختلفتان | 2 different backgrounds
3. ✅ استخدام الحلقات (Loops) | Use of loops
4. ✅ استخدام الشروط (If-Then) | Use of conditionals
5. ✅ التفاعل مع المستخدم (أزرار، لوحة المفاتيح) | User interaction
6. ✅ الأصوات أو الموسيقى | Sounds or music
7. ✅ المتغيرات (Variables) لحفظ النقاط أو التقدم | Variables for score/progress

**القيود | Constraints:**
- يجب أن تكون اللعبة قابلة للعب لمدة 2-3 دقائق على الأقل
- Game must be playable for at least 2-3 minutes
- يجب أن تحتوي على تعليمات واضحة | Must include clear instructions
- يجب استخدام 3 عناصر مصرية على الأقل (هرم، نيل، فرعون، إلخ)
- Must use at least 3 Egyptian elements

#### أفكار للمشاريع | Project Ideas
1. **لعبة جمع الكنوز**: جمع الآثار المصرية | Treasure collection game
2. **قصة تفاعلية**: استكشاف معبد قديم | Interactive story exploring a temple
3. **اختبار المعرفة**: أسئلة عن مصر القديمة | Knowledge quiz about Ancient Egypt
4. **لعبة المتاهة**: الهروب من الهرم | Maze game escaping the pyramid

### المستوى الثالث: المرحلة الإعدادية | Level 3: Preparatory Stage

#### المهمة الرئيسية | Main Task
**"محاكي الحضارة المصرية | Egyptian Civilization Simulator"**

كتابة برنامج Python يحاكي أحد جوانب الحضارة المصرية القديمة.

Write a Python program that simulates an aspect of Ancient Egyptian civilization.

#### خيارات المشروع | Project Options

**الخيار 1 | Option 1: لعبة نصية تفاعلية | Text-based Adventure Game**
- استكشاف هرم أو معبد
- اتخاذ قرارات تؤثر على القصة
- نظام نقاط أو صحة

**الخيار 2 | Option 2: آلة حاسبة الهيروغليفية | Hieroglyphic Converter**
- تحويل النص الإنجليزي إلى رموز هيروغليفية (Unicode)
- عرض الترجمة بشكل جميل
- حفظ وقراءة الملفات

**الخيار 3 | Option 3: نظام إدارة المتحف | Museum Management System**
- قاعدة بيانات للآثار المصرية
- إضافة، حذف، بحث عن القطع
- إحصائيات وتقارير

**الخيار 4 | Option 4: محاكي بناء الأهرامات | Pyramid Building Simulator**
- حساب الموارد المطلوبة
- محاكاة الوقت والعمال
- رسوم بيانية (optional: using matplotlib/turtle)

#### المتطلبات التقنية | Technical Requirements

**يجب أن يتضمن البرنامج | Program Must Include:**
1. ✅ Functions (دوال) - على الأقل 3 | At least 3 functions
2. ✅ Loops (حلقات) - for و/أو while
3. ✅ Conditionals (شروط) - if/elif/else
4. ✅ Data Structures - قوائم (lists) أو قواميس (dictionaries)
5. ✅ User Input/Output - تفاعل واضح مع المستخدم
6. ✅ Comments - تعليقات توضيحية باللغة العربية أو الإنجليزية
7. ✅ Error Handling - معالجة الأخطاء الأساسية (try/except)

**المكتبات المسموحة | Allowed Libraries:**
- Built-in modules: random, time, math, datetime
- File handling: os, json
- Optional: turtle (للرسم), matplotlib (للرسوم البيانية)

## 👥 لجنة التحكيم | Judging Panel

### التكوين | Composition
- رئيس قسم التكنولوجيا | Technology Department Head
- 2 معلمي تكنولوجيا | 2 Technology teachers
- ولي أمر متخصص في التقنية (اختياري) | Parent with tech background (optional)
- ممثل عن الإدارة | Administration representative

### سير العمل | Workflow
1. **التقييم الفردي**: كل عضو يقيم كل مشروع مستقلاً
   - **Individual Assessment**: Each member evaluates independently
2. **المناقشة الجماعية**: مقارنة التقييمات
   - **Group Discussion**: Compare evaluations
3. **الاتفاق**: الوصول لقرار نهائي
   - **Consensus**: Reach final decision
4. **الإعلان**: إعلان النتائج في حفل الختام
   - **Announcement**: Announce results at ceremony
