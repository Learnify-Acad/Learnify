# 🥇 CAMP-AI — AI Makers Camp
### *الأكثر طلباً في مصر 2025–2026*

| البند | التفاصيل |
|-------|----------|
| **العمر** | 10–16 سنة |
| **الأدوات** | Teachable Machine, Scratch AI extensions, Python (مستوى متقدم) |
| **السعر** | 2,500 ج / طالب |
| **الشعار** | "اصنع ذكاءً يفهم العالم من حوله" |

### المشروع النهائي (يختار الطالب)
1. **كاشف الآثار** — AI يتعرف على صور هيروغليف / أهرامات
2. **مساعد المتحف** — Chatbot يرد على أسئلة عن مصر القديمة
3. **النبات الذكي** — محاكاة جهاز يراقب صحة النبات (مفهوم IoT + AI)

### خطة اليوم التفصيلية

| اليوم | Mini-Lesson | Lab | Project % |
|-------|-------------|-----|-----------|
| 1 | ما هو AI؟ بيانات وتدريب | تجربة Teachable Machine | 10% |
| 2 | جمع البيانات + تصنيف | بناء dataset مصري | 25% |
| 3 | تدريب النموذج + اختبار | تحسين الدقة | 40% |
| 4 | ربط AI بتطبيق/لعبة | واجهة المستخدم | 55% |
| 5 | Alpha Demo | عرض + feedback | 65% |
| 6 | ميزات إضافية | أصوات / ردود / إحصائيات | 75% |
| 7 | تحسين الأداء | معالجة حالات الخطأ | 85% |
| 8 | فيديو + README | توثيق المشروع | 92% |
| 9 | Pitch training | تدريب 5 دقائق | 98% |
| 10 | **Demo Day** | عرض + جوائز | 100% |

### مخرجات التعلم
- فهم كيف يعمل AI (بدون رياضيات معقدة)
- بناء نموذج تصنيف بسيط
- ربط AI بمشروع تفاعلي
- عرض تقني أمام جمهور

**الشهادة:** Learnify AI Maker — Summer 2026

### المشروع النهائي

### المتطلبات التقنية (Checklist)
- [ ] 4 شخصيات (Sprites)
- [ ] 3 خلفيات
- [ ] Loops + Conditionals + Variables
- [ ] تفاعل لوحة مفاتيح/ماوس
- [ ] شاشة بداية + نهاية + تعليمات
- [ ] 3 دقائق لعب على الأقل

### أفكار جاهزة
1. **كنوز الهرم** — جمع + أعداء + متاهة
2. **حارس المعبد** — أسئلة + مستويات
3. **رحلة النيل** — قصة بخيارات
