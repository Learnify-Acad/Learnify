عن هذا المستوى | About This Level

مشاريع برمجية تفاعلية تربط بين التراث المصري والتكنولوجيا. يمكن للطلاب استخدام Scratch لإنشاء ألعاب وقصص تفاعلية، أو استخدام Python للطلاب الأقوياء في الصفين الخامس والسادس.

Interactive programming projects linking Egyptian heritage with technology. Students can use Scratch to create games and interactive stories, or Python for advanced Grade 5-6 students.

💻 متطلبات Scratch:
  • 3 شخصيات على الأقل (sprites)
  • خلفيتان مختلفتان
  • نظام نقاط أو تقدم
  • أصوات أو موسيقى
🐍 متطلبات Python:
  • 3 دوال على الأقل (functions)
  • استخدام القوائم أو القواميس
  • تعليقات توضيحية بالعربي/الإنجليزي
  • معالجة مدخلات المستخدم
